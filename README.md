# Development

## Prerequisites

- [yarn](https://yarnpkg.com/)

## Dependencies

Install all the dependencies with yarn.

```bash
$ yarn 
```

## Run

Run a local development instance.

```
$ yarn dev
```

# Build
Generate static files in the ```/dist``` folder

```
yarn build
```
